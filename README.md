# acif104-churn-grupo3

# Predicción de abandono de clientes

Proyecto académico de la asignatura ACIF104 - Aprendizaje de Máquinas.

## Objetivo

Analizar un dataset de clientes y desarrollar modelos de Machine Learning
para predecir el riesgo de abandono o churn.

## Estructura

- `data/raw`: datos originales.
- `data/processed`: datos depurados.
- `notebooks`: análisis y modelamiento.
- `src`: funciones y scripts.
- `figures`: gráficos obtenidos.
- `reports`: informes del proyecto.

## Modelos candidatos

- Regresión logística
- Random Forest
- XGBoost
- Modelos de Deep Learning
